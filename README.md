#  Personal Finance Calculator

A command-line Python tool to track monthly income and expenses using the **50/20/30 budgeting rule**.

---

##  What It Does

- Collects monthly income and categorizes expenses into:
  - **Fundamentals (50%)** — Rent, Groceries, Transport, Credit Card, Miscellaneous
  - **Fun (20%)** — Netflix, Spotify, Shopping, Eating Out
  - **Future You (30%)** — Stocks, Bonds, Mutual Funds, Index Funds, Gold ETF, Emergency Fund
- Compares your spending against the 50/20/30 targets
- Calculates your net savings
- Saves a detailed report to `finance_report.txt` after every month
- Supports tracking **multiple months** in one session

---

##  Requirements

- Python 3.6 or higher
- No external libraries — uses Python standard library only

---

##  How to Run

**Step 1 — Clone the repository**
```bash
git clone https://github.com/your-username/personal-finance-calculator
cd personal-finance-calculator
```

**Step 2 — Run the script**
```bash
python finance_calculator.py
```

> On some systems use `python3` instead of `python`

---

##  Output

A file called `finance_report.txt` is automatically created in the same folder after each session. It stores a formatted summary of every month you track.
